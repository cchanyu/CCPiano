### Project: Roblox Concert Hall
### Program Name: Piano related scripts and parts

### Description: 
``` 
Game Link: https://www.roblox.com/games/5527931344/KillerMachine-Piano
Interactive live piano coded in lua, can be played inside Roblox games.
This piano is following standard MIDI keyboard format.

Transpose = pitch
SoundFont = type of instrument
Caps = capslock boolean
Exit = quits piano
Keys = plays a sound
```
